# GitHub Learning Journal

## 專案目的
這是一個 GitHub 新手練習專案，用來學習：
- 建立 Repository
- 編輯 README
- 新增檔案與資料夾
- 使用 Commit 紀錄學習過程

## 專案結構
github-learning-journal
├─ README.md
├─ resources.md
├─ week1/notes.md
└─ week2/notes.md

## 學習紀錄
### Week 1
- 成功建立第一個 GitHub 專案
- 學會新增檔案與 Commit

### Week 2
- 能獨立建立資料夾與 Markdown 檔案
- 開始閱讀他人 GitHub 專案

## 備註
本專案為個人 GitHub 學習用途。
